## 1. System Requirements
### Operating systems

The code has been tested on the following operating systems:

- macOS 15 (Sequoia)
- Windows 11

The code should run on other modern macOS or Windows systems with compatible MATLAB versions, but this has not been explicitly tested.

### Software dependencies
- MATLAB R2024a (MathWorks, Natick, MA)

### Hardware requirements
- No non-standard hardware is required.
- The code was developed and tested on standard desktop and laptop computers.

## 2. Installation guide
### Installation instructions

## Notes on data availability
The original clinical data used in the manuscript contain protected health information and cannot be distributed. As a result, no demo dataset is included in this repository. The code is provided to allow inspection of the algorithms, implementation details, and analysis pipeline.
